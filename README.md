# is-html-tag
Function to check whether tag is a proper html tag

## How to use it

```javascript
    var isHTMLTag = require('is-html-tag'); 
    // your beautiful code
    ..
        if(isHTMLTag("div"){
            // whatever you want to do with your html tag 
        }else{
            // whatever you want to do with your non html tag
        }
```
